# Weekly Letter Syndication Playbook

## The rule
Write once. Repurpose cleanly. Never spam.

## Sunday 7 PM PT — Substack
Publish the full letter.

## Sunday 8 PM PT — LinkedIn Article
Copy full letter into a LinkedIn Article. Change the first paragraph to a LinkedIn-native intro ("A quick operator note for anyone following what we're building at Angel Precision."). Add a professional image or dashboard screenshot.

## Monday 7 AM PT — Medium
Copy full letter into Medium. Different intro: more essay-flavored. Tag: automation, systems, trading, founder.

## Monday 9 AM PT — X thread
Pull 5–8 posts from the letter. Structure:
1. Hook — a sentence from the letter
2. The incident
3. The lesson
4. The operating number
5. The founder line
6. What's next
7. CTA

## Tuesday 8 AM PT — Facebook note
180-word version. Conversational. One paragraph + CTA.

## Tuesday 10 AM PT — Warm contact email
BCC list. Short intro: "Sharing this week's Operator's Letter — always happy to talk." Link to Substack.

## Optional Wednesday — Reddit
ONLY if one technical insight stands alone. Post manually in a real subreddit (algotrading, options, systems engineering). No links to Substack unless the sub allows it.

## Never
- Auto-syndicate to Reddit
- Cross-post identical copy to multiple subreddits
- Send the same intro across platforms
- Use automated Twitter thread generators without editing

## Monthly
Compile the month's four letters into a "Monthly Operator Digest" — send as a premium-feel PDF to warm contacts and prospects. This becomes sales collateral.
