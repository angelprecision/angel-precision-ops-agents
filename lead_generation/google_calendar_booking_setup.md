# Google Calendar Booking Setup

## Fast path tonight
If you are using Google Calendar, create one booking page and use it everywhere.

## Steps
1. Open Google Calendar.
2. Click Create -> Appointment schedule.
3. Create `Angel Precision Fit Call`.
4. Duration: 20 minutes.
5. Buffer: 10 minutes.
6. Availability: Tue-Fri, limited windows only.
7. Add description:
   - short intro
   - this is a fit/qualification call
   - no guarantees, no trade advice, no pressure
8. Add booking form questions:
   - account size band
   - trading experience
   - goals
   - risk tolerance
   - where they found you
9. Copy booking URL.
10. Add to env, email templates, waitlist page, Substack, LinkedIn bio, X bio, and confirmation emails.
