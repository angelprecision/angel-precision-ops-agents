# angel-precision-ops-agents
OPS Agents

Operational automation system for Angel Precision.

This repo contains:
- n8n workflow plans
- agent prompts
- content approval rules
- command center dashboard
- operator letters
- deployment checklists
- Supabase schema

Trading bot code lives in the separate Angel Precision bot repo.

Rule:
Agents draft, log, route, and summarize.
They do not make irreversible business decisions without founder approval.
