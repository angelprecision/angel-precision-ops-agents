# Platform Style Guide

## Brand voice
- calm
- technical
- transparent
- serious
- ambitious
- never hypey

## Core themes
- execution quality
- risk controls
- monitoring
- kill switches
- health dashboards
- rejection logic
- live refinement
- founder journey
- operational visibility

## Avoid phrases
- guaranteed returns
- passive income
- easy money
- set and forget
- can't lose
- low risk high reward

## Preferred language
- controlled deployment
- operator oversight
- predefined rules
- execution discipline
- monitored system
- risk-managed process
- fit call
- beta refinement
