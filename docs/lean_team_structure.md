# Lean Team to Deploy While You Build

## 1) Founder / Operator (you)
Owns:
- final approvals
- pricing decisions
- sales calls
- ops review
- client trust decisions

## 2) Automation Engineer
Owns:
- Supabase schema
- n8n workflows
- Gmail OAuth
- webhook setup
- calendar integration
- kill-switch and logs

## 3) Content Operator
Owns:
- review content queue
- schedule approved posts
- ensure platform formatting is correct
- maintain editorial calendar

## 4) Sales Coordinator
Owns:
- verify lead data
- manage follow-up queue
- book calls
- keep statuses clean

## 5) Ops Analyst
Owns:
- review daily ops reports
- tag recurring errors
- escalate warnings/criticals
- maintain incident notes

## 6) Compliance Reviewer (fractional / advisor)
Owns:
- review templates touching price, risk, performance, and client claims
- approve safe language

## Minimal today if solo
If solo today, collapse roles like this:
- You = Founder + Sales + final approvals
- One builder = Automation Engineer
- One helper/VA = Content Operator + Sales Coordinator
