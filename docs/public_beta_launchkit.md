# Public Beta Launch Kit

## Positioning line
Angel Precision is accepting controlled beta interest from people who want a monitored options execution and risk-management system — not signals, hype, or guaranteed-return promises.

## CTA
Apply for beta access or book a short fit call.

## Where to place this
- X bio
- LinkedIn featured link
- Facebook profile/about
- Substack footer
- Medium profile
- Landing page hero
- Email signature
- Discord/community bio
- Every content post CTA (softly)

## Bio template (adapt per platform)
```
Building Angel Precision — automated options execution + risk management.
Monitored. Controlled. No hype.
Accepting limited beta interest.
[booking link]
```

## Waitlist page CTA copy
```
Apply for controlled beta access
```

NOT:
```
Sign up now
Join the revolution
Start making money
```

## Fit call framing
The call is for:
- understanding account size and goals
- assessing risk tolerance
- explaining how the system works operationally
- confirming mutual fit before any payment

The call is NOT:
- a sales pitch
- a demo of returns
- a guarantee conversation
- pressure to commit immediately

## What to say when asked about performance
```
Trading involves risk. Angel Precision is built to control execution, manage risk, and monitor system health through predefined rules. The goal is disciplined, observable execution — not promises. I can walk you through how the system handles entries, exits, rejections, and monitoring on a call.
```

## Launch sequence
1. Update all bios with positioning line + booking link
2. Publish Substack welcome post
3. Post one LinkedIn post about beta access
4. Post two X posts about controlled deployment
5. Post one Facebook note
6. Hold Reddit for technical-only posts
7. Email existing warm contacts with the waitlist link
8. Ask 3-5 personal contacts for referrals

## Pricing posture on calls
- Monthly operator fee starting at $1,500/mo
- Scaled by account size and risk profile
- Beta window may offer adjusted onboarding
- Always confirm fit before taking payment
- Never promise specific returns or timelines
