# Content Approval Checklist

Before approving any post, check:

## Hard blocks
- Does it guarantee returns?
- Does it promise profits?
- Does it imply risk-free trading?
- Does it invent client results?
- Does it sound like a signals room?
- Does it sound desperate or hypey?

If yes to any: reject and rewrite.

## Trust checks
- Is the post specific?
- Does it sound calm and serious?
- Does it reinforce monitoring, discipline, execution, and risk control?
- Does it fit founder/operator voice?

## Platform checks
- X: concise and sharp
- LinkedIn: clear and professional
- Facebook: conversational
- Reddit: technical and non-promotional
- Medium: structured and useful
- Substack: personal + substantive

## Final action
- Approve
- Edit
- Reject
- Hold for later
