# Daily Cadence Scorecard

## Purpose
Operational rhythm, not motivation. You are compounding trust, proof, leads, content, and ops maturity every day.

## Daily targets
```
1 founder post (LinkedIn or Substack)
2 X posts
5 outreach drafts generated
5 follow-ups sent or drafted
1 ops note added
1 dashboard/system refinement
1 proof artifact saved to vault
```

## Dashboard display
```
Founder post:         ✅ / ❌
X posts:              0/2
Outreach drafts:      0/5
Follow-ups:           0/5
Ops note:             ✅ / ❌
Dashboard refinement: ✅ / ❌
Proof artifact:       ✅ / ❌
```

## How to track
Insert one row per day into `daily_cadence_logs`.
Update throughout the day as tasks complete.
Dashboard view `dashboard_cadence_today` shows current status.

## Weekly review
Every Sunday night:
- How many days hit 7/7?
- Which targets are consistently missed?
- What blocked completion?
- Adjust targets if needed (up or down)

## Why this matters
You are not "starting over" each day. You are compounding:
- trust
- proof
- leads
- content
- ops maturity

Missing one day is fine. Missing a pattern is a problem.
